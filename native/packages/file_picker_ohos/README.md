# file_picker_ohos

This package vendors the OHOS native plugin registration from
`CPF-Flutter/fluttertpc_file_picker` at commit
`31cc1f314557517e65694ac1744567670f0829c2`.

Only the `ohos` platform is declared in `pubspec.yaml`. The upstream fork also
contains Android/iOS/macOS implementations whose Java classes duplicate the
official `file_picker` package on Android release builds.
